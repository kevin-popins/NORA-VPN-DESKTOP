# Twemoji graphics attribution

The PNG graphics in this folder are Twemoji artwork, sourced from the
`twemoji-emojis` 14.1.0 package. They are used locally by NORA VPN so emoji
inside subscription names and provider announcements render consistently in
full colour without a network request or a dependency on the user's Windows
emoji-font support.
