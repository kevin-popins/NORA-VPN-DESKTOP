# Security policy

Please do not open public issues with connection keys, subscription URLs,
server addresses, logs, screenshots containing private data, or vulnerability
details.

For a responsible disclosure, contact **@RunwayResearch** on Telegram. Include
the affected version, a minimal reproduction, and an impact summary. We will
acknowledge a report and coordinate a fix before public disclosure.

## Local-data rule

NORA VPN stores imported subscriptions and credentials locally. They are not
part of this repository and must never be attached to an issue or pull request.
